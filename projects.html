  1. Spring MVC for ISBN Validation
    This Java project implements the Spring MVC framework and uses the tomcat server to validate 10 or 13 digits ISBN. I used Eclipse as an IDE. You can download the project and import it into Eclipse to run.

  2. Abstract and Interface
    This Java project implements Abstract and Interface classes along with the concept of Polymorphism.

  3. US Temperature Map
    This JavaScript project uses RESTful API from darksky.net. The US map is created using SVG (Scalable Vector Graphics), a vector-based graphics. JavaScript is used to pull temperature data and to display state color.

  4. Tictac Game
    A basic TicTac game using JavaScript and HTML. The JavaScript code is a modified version of Cory Fogliani.
